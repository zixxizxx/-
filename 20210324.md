- acl = access control list 控制訪問列表
先把 IP 分類
內置acl
```

- IP、網路 l.2.3.4、l.2.3.4/24
- any
- none
-localhost 
-localnet 智能DNS的區網
```
自定義acl
```
族群
acl"名稱1"{
    ip1 ; ip2; īp3 ;
    };
acl"名稱2"{
    "名稱1"；
    10.0.16/24;網路位址
};
```
使用acl I
```
allow_update { "名稱" ;};
allow_transfer { "名稱" ;};
```
